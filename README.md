# Student Marks Analyzer

A simple Java console application that analyzes student marks.

## Features

- Store student marks
- Find highest marks
- Find lowest marks
- Calculate average marks
- Display grades

## Technologies Used

- Java
- Scanner
- Arrays
- Loops
- Conditional Statements

## Sample Output

```
Enter number of students:
5

Enter marks:
85
92
67
74
56

Highest Marks = 92
Lowest Marks = 56
Average = 74.8

Grades

Student 1 : B
Student 2 : A
Student 3 : D
Student 4 : C
Student 5 : F
```
